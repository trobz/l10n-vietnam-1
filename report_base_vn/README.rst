Report Base Vietnam
============================

Support function in generate report
-----------------------------------
* convert_amount: Convert Amount from currency A to currency B depends on Date
* addr_get: Get all Add information of partner input
* bank_get: Get all Bank information of partner input
* amount_to_text_vi: Convert amount to text in Vietnamese

Print Invoice PDF Report by pass https authentication restriction
-----------------------------------
* Set param: report.url
* Value: local_xmlrpc_interface:xmlrpc_port
